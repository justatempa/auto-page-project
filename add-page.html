如何添加页面

添加新页面的步骤

步骤 1：创建 HTML 文件

在项目根目录下创建一个新的 HTML 文件，例如 new-page.html

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">

    
    <meta http-equiv="Content-Security-Policy" content="default-src 'self';">
    <meta http-equiv="X-XSS-Protection" content="1; mode=block">
    <meta http-equiv="X-Frame-Options" content="DENY">
    <meta http-equiv="X-Content-Type-Options" content="nosniff">

    <title>新页面</title>
    <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/twitter-bootstrap/4.5.3/css/bootstrap.min.css">

    <style>
        /* 添加自定义样式 */
    </style>
</head>
<body>
    <div class="container mt-4">
        <h1>新页面</h1>
        <p>这是一个新的功能页面</p>

        <!-- 返回主页按钮必须包含 -->
        <a href="index.html" class="btn btn-primary">返回主页</a>
    </div>

    <script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
    <script src="https://cdnjs.cloudflare.com/ajax/libs/twitter-bootstrap/4.5.3/js/bootstrap.min.js"></script>
</body>
</html>
步骤 2：更新 pages.json

在 pages.json 文件中添加新页面的配置

[
    {
        "title": "关于我们",
        "description": "了解我们的项目背景",
        "file": "about.html",
        "image": "https://via.placeholder.com/400x200?text=About+Us"
    },
    {
        "title": "新页面",
        "description": "这是一个新的功能页面",
        "file": "new-page.html",
        "image": "https://via.placeholder.com/400x200?text=New+Page"
    }
]

配置项说明

配置项 说明 是否必填
title 页面标题（显示在卡片上） 是
description 页面简短描述（显示在卡片上） 是
file HTML 文件路径 是
image 卡片图片 URL 否（默认使用占位图片）

安全提示

  • 确保所有页面都包含安全头部
  • 不要在页面中包含未经验证的用户输入
  • 使用 HTTPS 部署到服务器
  • 不要在页面中包含恶意脚本
